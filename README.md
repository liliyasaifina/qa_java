# sprint_6
## Автоматизированные тесты для Программы учета животных

### Описание

Набор автоматизированных тестов на Java для проверки классов Feline, Cat и Lion:
* тестирование методов классов Feline, Cat и Lion;
* использование набора данных для тестирования класса Lion.

### Технологии, зависимости и плагины
* Java 11
*  Maven
* JUnit 4.13.2
* Mockito 4.8.0
*  jacoco-maven-plugin 0.8.7
* maven-surefire-plugin 2.22.2

### Запуск тестов

```shell
mvn clean test
```
